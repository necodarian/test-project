1: for å sjekke endringer i den mappen man jobber på: "git status"

2: for å adde en fil til git : "git add <filename>"

3: for å legge til en endring man har gjort i en fil som allerede er addet til git: "git commit -m(message) "message""

4: for å sende videre til github server "git push" 

5: for å hente ut nyeste fra github-server "git pull"

6: create new branch " git checkout -b <branch-name>"

7: For å sjekke antall branches og hvilke branch som er active " git branch"

8: for å slette en branch "git branch -d <branch-name>"

9: for å forcefully slette en branch "git branch -D <branch-name>"

10: for å skifte mellom branches " git checkout <branch-name>"
